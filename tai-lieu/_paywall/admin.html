Quản lý Paywall

Chọn tier và số mục miễn phí cho từng tài liệu. Click Lưu config để tải file JSON rồi upload đè lên tai-lieu/_paywall/content-config.json trong repo GitHub. Sau khi push, các trang sẽ tự đọc config mới.

Cách triển khai:
1. Sửa các tier / free units ở bảng trên
2. Bấm Lưu config — trình duyệt tải file content-config.json về máy
3. Upload file đó đè lên tai-lieu/_paywall/content-config.json trong repo luminhtri0-hash/trilu-website trên GitHub
4. GitHub Pages tự deploy trong 1-2 phút — các trang content sẽ áp dụng phân quyền mới
